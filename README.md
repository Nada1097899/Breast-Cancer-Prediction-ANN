#  Breast Cancer Prediction using ANN

A Deep Learning project that predicts whether a tumor is Malignant or Benign based on medical features, achieving high diagnostic accuracy.

## Project Overview
This project utilizes an Artificial Neural Network (ANN) to classify breast cancer cases. By analyzing features like cell radius, texture, and perimeter, the model assists in early and accurate diagnosis.

##  Tech Stack
* **Language:** Python
* **Deep Learning Framework:** Keras / TensorFlow
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn.
* **Architecture:** Multi-layer Perceptron (ANN) with Dropout for regularization.

## Key Results
The model performed exceptionally well on the test dataset:
* **Training Accuracy:** ~99%
* **Validation Accuracy:** High stability shown in Accuracy/Loss curves.
* **Optimization:** Used 'Adam' optimizer and 'Binary Crossentropy' loss function.

## Visualizations
The project includes:
- Correlation heatmaps of medical features.
- Training vs. Validation loss/accuracy plots.
- Confusion matrix for performance evaluation.
